## About The Project
My Duties:
• Lead a team of software developer and built a clean and organized code.
• Designed and built a car and developed embedded system using the Arduino micro-controller board .
• Designed, developed and implemented an Arduino software that has the capability to:
	- Retrieve and analyze data from the QTR-8A and HC-SR04 sensors.
	- Manage the power of the rotors
 
Components used:
 - Arduino Genuino Uno
 - 2 chassis
 - Wheels x4
 - Rotors x4
 - L298N sensor
 - Array of sensors (6)
 - 2 ultrasound sensor
 - Battery x8
